# 基金涨跌查看工具

一个实时查看基金估值变化的网页工具，解决支付宝无法查看当天涨跌的问题。

## 功能特点

- 🔍 **基金搜索**：输入基金代码查询实时估值
- ⭐ **收藏列表**：收藏关注的基金，方便查看
- 🔄 **自动刷新**：每30秒自动更新数据
- 📱 **响应式设计**：支持手机和电脑访问
- 💾 **本地存储**：收藏列表保存在浏览器中
- 🌙 **深色模式**：自动跟随系统深色模式
- 📲 **PWA 支持**：可添加到手机主屏幕

## 使用方法

### 1. 在线访问

部署后直接访问 Vercel 分配的域名即可使用。

### 2. 本地开发

直接在浏览器中打开 `index.html` 文件即可使用。

或者使用本地服务器（推荐）：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server

# 使用 VS Code 的 Live Server 插件
# 右键 index.html -> Open with Live Server
```

然后访问 `http://localhost:8000`

### 2. 搜索基金

1. 在搜索框输入基金代码（如：110011）
2. 点击"查询"按钮或按回车键
3. 查看基金估值信息
4. 点击"收藏"按钮添加到收藏列表

### 3. 管理收藏

- 收藏的基金会显示在"我的收藏"区域
- 点击"刷新"按钮手动更新数据
- 点击"删除"按钮移除不需要的基金

## 数据来源

使用天天基金网的公开 API 获取基金估值数据。

**注意**：
- 估值数据仅供参考，非实时净值
- 数据更新频率为每30秒一次
- API 有请求频率限制，请勿过于频繁刷新

## 部署到 Vercel

### 方法一：通过 Vercel CLI

1. 安装 Vercel CLI
```bash
npm i -g vercel
```

2. 登录 Vercel
```bash
vercel login
```

3. 部署项目
```bash
cd fund
vercel
```

4. 按照提示操作，完成后会得到一个域名

### 方法二：通过 Vercel 网站

1. 将代码推送到 GitHub 仓库
2. 访问 [vercel.com](https://vercel.com)
3. 点击 "New Project"
4. 导入你的 GitHub 仓库
5. 点击 "Deploy"
6. 等待部署完成，获取域名

### 自定义域名（可选）

1. 在 Vercel 项目设置中点击 "Domains"
2. 添加你的自定义域名
3. 按照提示配置 DNS 记录
4. 等待 DNS 生效
